use_namespace

The use_namespace name TCL command searches for all TCL commands of the form name.x, and creates a new TCL command x that calls name.x. This allows a simple ``dump'' of Eco Lab model methods and instance variables into TCL's current namespace. The term ``namespace'' in use_namespace is unrelated to TCL's namespace concept.

Note that use_namespace does not override an any existing commands in the current namespace. Imagine what havoc an instance variable called ``proc'' would make! Thus it may be necessary to refer to an instance variable or method by its fully qualified name fro TCL scripts to get the desired behaviour.
